# U-map - Carte Interactive du Campus UAC

Une application web moderne et interactive pour explorer le campus de l'Université d'Abomey-Calavi (UAC) au Bénin.

## 🌟 Fonctionnalités

- 🗺️ **Carte Interactive** : Explorez le campus avec Leaflet.js
- 📍 **Localisation des Infrastructures** : Plus de 100 lieux référencés sur le campus
- 🎯 **Itinéraires** : Calculez des itinéraires entre différents points du campus
- 🌓 **Mode Sombre/Clair** : Interface adaptable selon vos préférences
- 📱 **Responsive** : Fonctionne sur tous les appareils
- 🎨 **UI/UX Premium** : Design moderne avec animations fluides
- 💾 **Historique Local** : Sauvegarde de vos lieux visités

## 🛠️ Technologies Utilisées

- **Vue.js 3** (Composition API)
- **Vue Router 4** - Navigation
- **Pinia** - Gestion d'état
- **Leaflet.js** - Cartographie interactive
- **TailwindCSS** - Styling
- **AOS** - Animations au scroll
- **SwiperJS** - Carrousels d'images
- **Iconify** - Icônes

## 📦 Installation

```bash
# Cloner le dépôt
git clone https://github.com/VOTRE_USERNAME/U-map.git

# Accéder au dossier
cd U-map

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
```

## 🚀 Utilisation

```bash
# Développement
npm run dev

# Build pour production
npm run build

# Prévisualiser le build
npm run preview
```
